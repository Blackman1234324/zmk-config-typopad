# TypoPad

TypoPad is my custom 7-key macropad project built with **NiceNano**. It's compact, programmable, and comes with **potentiometer support** for analog input control.

## Features

- 7 programmable keys
- Potentiometer support
- RGB LED support (optional)
- Compact design with 3D printable case
- Easy firmware flashing with QMK/ZMK

## Hardware

- **Microcontroller:** NiceNano  
- **Switches:** Mechanical switches of your choice  
- **Potentiometer:** 10kÎ© (for volume, scroll, or custom mapping)  
- **LEDs:** WS2812 (optional)  
- **Case:** 3D printable
